# jonis-catering

Repository und Obsidian-Vault für Jonis Catering.

## Struktur

- `index.html` — Website (jonis-catering.de)
- `CLAUDE.md` — System-Workflow für Claude-Sessions (wird automatisch geladen)
- `JONIS-OS/` — Obsidian-Vault mit allen Bereichen
  - Einstieg: `JONIS-OS/00_START_HIER.md`

## Bereiche

Catering · Website · Marketing · Equipment · Solar · Finanzen · Automationen
Querschnitt: Prompts · Chat-Summaries · SOPs

## Nutzung

- **Obsidian**: `JONIS-OS/` als Vault öffnen
- **Claude**: `CLAUDE.md` definiert Workflow, Tonalität und Konventionen
- **Git**: alle Änderungen werden versioniert
